# GEO Writer - Sistema de Generación Automática con IA

Sistema profesional de generación automática de contenido con IA para WordPress.

## Descripción

GEO Writer es un plugin de WordPress que permite generar contenido automático de alta calidad utilizando inteligencia artificial. Incluye gestión de campañas, generación de colas de publicación, integración con API de IA, y sistema de licencias.

## Características principales

- ✅ Generación automática de posts con IA
- ✅ Sistema de campañas y gestión de colas
- ✅ Integración con múltiples modelos de IA
- ✅ Sistema de licencias integrado
- ✅ Tutoriales interactivos con Shepherd.js
- ✅ Interfaz moderna y profesional
- ✅ Compatible con PHP 7.4+

## Requisitos

- WordPress 5.8 o superior
- PHP 7.4 o superior
- cURL habilitado
- Licencia válida de Bocetos Marketing

## Instalación

1. Descarga el plugin desde [Releases](https://github.com/bocetosmarketing/geowriter/releases)
2. Sube el archivo ZIP a través de WordPress (Plugins > Añadir nuevo > Subir plugin)
3. Activa el plugin
4. Configura tu licencia en el menú GEO Writer

## Actualizaciones automáticas

El plugin se actualiza automáticamente desde este repositorio de GitHub. WordPress te notificará cuando haya una nueva versión disponible.
