# Conversa - Chat con esteroides

Sistema avanzado de chatbot con IA para WordPress, integraciones con Telegram y base de conocimientos.

## Descripción

Conversa es un plugin de WordPress que proporciona un chatbot inteligente con capacidades avanzadas de IA, integración con Telegram, gestión de leads, base de conocimientos y múltiples fuentes de datos.

## Características principales

- ✅ Chatbot con IA integrado en tu sitio web
- ✅ Integración con Telegram
- ✅ Base de conocimientos (Knowledge Base)
- ✅ Gestión de leads y estadísticas
- ✅ Crawler para fuentes de datos externas
- ✅ Inyección de contenido en páginas específicas
- ✅ Tutoriales interactivos con Shepherd.js
- ✅ Interfaz personalizable con múltiples paletas de colores

## Requisitos

- WordPress 5.0 o superior
- PHP 7.4 o superior
- Licencia válida de Bocetos Marketing

## Instalación

1. Descarga el plugin desde [Releases](https://github.com/bocetosmarketing/conversa-bot/releases)
2. Sube el archivo ZIP a través de WordPress (Plugins > Añadir nuevo > Subir plugin)
3. Activa el plugin
4. Configura el chatbot en Conversa > Configuración

## Actualizaciones automáticas

El plugin se actualiza automáticamente desde este repositorio de GitHub. WordPress te notificará cuando haya una nueva versión disponible.

## Módulos incluidos

### Chat
Sistema de chat frontend con IA y múltiples personalizaciones.

### Telegram
Integración completa con Telegram Bot API para recibir notificaciones y mensajes.

### Knowledge Base (KB)
Base de conocimientos editable para entrenar al chatbot.

### Leads
Gestión y seguimiento de conversaciones y leads capturados.

### Estadísticas
Dashboard con métricas de uso y conversaciones.

### Crawler
Sistema para importar contenido de URLs permitidas como fuente de datos.

### Inject
Inyectar el chatbot en páginas específicas del sitio.
